# Jogo do Número Secreto

Projeto desenvolvido em JavaScript com o objetivo de praticar lógica de programação e interação com o usuário.

O jogo gera um número aleatório que o jogador deve adivinhar. Antes do início da partida, é possível selecionar o nível de dificuldade (Fácil, Médio ou Difícil), alterando o intervalo de números possíveis e tornando o desafio mais interessante. Durante o jogo, o sistema fornece dicas indicando se o número secreto é maior ou menor que o valor informado pelo jogador.

Além dos requisitos básicos, foram implementadas melhorias por iniciativa própria, como a seleção de dificuldade, validação de entradas utilizando `isNaN()` para impedir valores não numéricos, tratamento de dificuldades inválidas, verificação de números fora do intervalo permitido e contagem de tentativas com exibição correta no singular e plural.

## Conceitos praticados

* Variáveis e constantes
* Estruturas condicionais (`if`, `else if`, `else`)
* Laços de repetição (`while`)
* Operadores lógicos (`||`)
* Operador ternário
* Geração de números aleatórios (`Math.random()` e `Math.floor()`)
* Conversão de tipos com `Number()`
* Validação de dados com `isNaN()`
* Interação com o usuário através de `prompt()`, `alert()` e `console.log()`

Este projeto faz parte da minha jornada de aprendizado em JavaScript e demonstra a aplicação prática dos conceitos estudados, incluindo funcionalidades extras desenvolvidas para aprimorar a experiência do usuário.
